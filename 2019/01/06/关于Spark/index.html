关于Spark

一月 06, 2019


首先，Spark是什么，官方定义: Apache Spark is a fast and general-purpose cluster computing system.
It provides high-level APIs in Java, Scala, Python and R, and an optimized engine that supports general execution graphs.
It also supports a rich set of higher-level tools including Spark SQL for SQL and structured data processing,
MLlib for machine learning, GraphX for graph processing, and Spark Streaming.

Apache Spark是一个高效通用的，基于集群的计算系统。它提供了基于Java，Scala和R语言的高阶API，以及支持通用执行图的优化引擎。
它还支持一组丰富的更高级别的工具，包括Spark SQL和结构化数据的处理，MLlib用于机器学习，GraphX用于图形处理和Spark Stream。
